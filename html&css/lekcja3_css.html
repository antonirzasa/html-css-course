Typy stylów CSS

  • Arkusz ZEWNĘTRZNY (umieszczany w osobnym dokumencie z rozszerzeniem .css)
  • Arkusz WEWNĘTRZNY (umieszczany w pliku HTML w znaczniku <head>, wewnątrz znacznika <style> )
  • Arkusz LOKALNY (umieszczany wewnątrz konkretnego znacznika)

Hierarchia w CSS

Jeżeli pojawią się wielokrotnie zdefinfowane wartości dla tej samej właściwości selektora, obowiązuje następująca hierarchia przykrywania:

  1. Styl lokalny
  2. Rozciągnięcie stylu <span>
  3. Wydzielone boki <div>
  4. Wewnętrzny arkusz stylów
  5. Zewnętrzny arkusz stylów
  6. Import arkusza stylów
  7. Domyślne ustawienia przeglądarki
